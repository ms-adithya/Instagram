# Instagram-like Interface

This project is a responsive Instagram-like interface built using **HTML** and **CSS**. It includes a sidebar, stories section, posts section, and a footer, all styled to mimic the layout and functionality of Instagram.

## âœ¨ Features

- **Responsive Design**: Adapts to different screen sizes (desktop, tablet, and mobile).
- **Sidebar Navigation**: Includes icons and labels for navigation.
- **Stories Section**: Displays user stories with circular images.
- **Posts Section**: Mimics Instagram posts with captions, likes, and comments.
- **Footer**: Contains user suggestions and links, hidden on smaller screens for better usability.

---

## ğŸ“± Responsive Design

The layout is fully responsive and adjusts based on the screen size:

- **Large Screens (Desktop)**: Full layout with sidebar, stories, posts, and footer.
- **Medium Screens (Tablets)**: Footer is hidden, and the layout adjusts for smaller widths.
- **Small Screens (Mobile)**: Sidebar and stories are resized, and posts take full width.
